## 1. Markedsbevegelse: "Vibe CEO" og One-Person Companies

Det eksisterer allerede et navn på dette mønsteret: **"One-Person Company"** eller **"Vibe CEO"**.

Enkeltpersoner kjører nå selskaper med milliomsomsetning ved å orkestrere AI-agenter istedenfor ansatte. Typisk stack:

| Funksjon | Verktøy |
|---|---|
| Koding | Cursor, Claude Code, Copilot |
| Design | Canva AI, Midjourney |
| Innhold/markedsføring | Descript, Opus Clip |
| Automatisering | Zapier, Make, n8n |
| Kundeservice | Intercom Fin, interaktive AI-agenter |
| Nettsider | Framer, Webflow, Vercel |

Kostnad for hele stacken: **$300–500/mnd** — mot ansatte som koster 500k+ i året.

**Norsk marked**: Implement Consulting Group (rapport 2025, Google-finansiert) anslår at AI kan skape **33.000 nye høyverdige arbeidsplasser** i Norge og bidra med **45 mrd NOK årlig** til økonomien. Det er et modent marked for nettopp dette.

## 2. Best passende forretningsmodeller

Basert på profilen (teknisk, norsk, .NET-bakgrunn, "CEO med agenter") — her er de tre sterkeste banene:

### A) AI-integrasjons- og automasjonsbyrå

**Hva**: Tobias selger timer/avtaler der han bygger AI-agenter og automasjoner for norske SMBer og scale-ups.

**Hvorfor det passer**:
- .NET-bakgrunn = teknisk troverdighet i enterprise/mid-market
- Norsk marked = forstår kunden, språk, kultur
- "CEO + agenter"-modellen er eksakt hvordan dette fungerer: Tobias selger, agenter bygger
- Lav startkostnad, høy margin
- Norsk SMB-marked er forsinket på AI-adopsjon = mulighet nå

**Revenue model**: Timepriser 1.500–2.500 kr/time, eller fastpris-prosjekter 50k–500k.

**Agent Leverage**: Flere kunder samtidig uten å ansette.

---

### B) Norsk AI-produkt (vertical SaaS)

**Hva**: Bygger et AI-produkt rettet mot en spesifikk norsk bransje (f.eks. helse, bygg, fiskeria, eiendom).

**Eksempler på vertical AI**:
- AI-agent for regnskapsførere (bokføring, rapportering)
- AI-agent for prosjektledere i bygg (dokumentasjon, rapportering, HMS)
- AI-agent for helsesektoren (pasientkommunikasjon, journalstøtte)

**Hvorfor det passer**:
- .NET = kan bygge robuste produkter
- "CEO med agenter" = kjapp time-to-market uten stort team
- Nisjemarked = mindre konkurranse, høyere priser

**Revenue model**: SaaS-abonnement 2.000–20.000 kr/mnd per kunde.

**Agent Leverage**: Én kodebase, ubegrenset skalering.

---

### C) AI-coaching/kurs for norske teknisk ledere

**Hva**: Tobias selger opplæring, workshops og rådgivning til norske bedrifter som vil forstå agentic AI.

**Hvorfor det passer**:
- Krever ingen ansatte — kun Tobias + innhold laget av agenter
- Passer godt med .NET-consultant-rykte
- Markedet er desperat etter kompetanse

**Revenue model**: Kurs 5.000–25.000 kr, workshops 15.000–50.000 kr, rådgivning 2.500–5.000 kr/time.

**Agent Leverage**: Max — kan selge async-kurs til ubegrenset antall samtidig.

## 3. Anbefalt retning

**Sterkeste trekk: Banen A (AI-integrasjonsbyrå) med et langsiktig mål om å spinne ut et produkt.**

**Hvorfor**:
1. Kan starte umiddelbart med eksisterende kontaktnettverk (Bravonet-kunder? kolleger?)
2. Cash-flow fra dag én (timepris-modell)
3. Bygger erfaring og case studies som senere blir produktet
4. Passer profilen: "CEO som leder agenter" — Tobias selger og styrer, agenter bygger

## 4. Handlingsplan

### Fase 1: Forberedelse (1–2 mnd)
- Sett opp "huset" (enkeltperson AS, konto, faktureringssystem)
- Definer 2–3 kjerneagentverktøy (foreslått: n8n + Claude/Copilot + Make)
- Lag en enkel portfolio-nettside
- Sett opp et CRM (HubSpot free tier)

### Fase 2: Første kunder (mnd 2–4)
- Bruk eksisterende nettverk aktivt
- Tilbud: "Jeg bygger deres første AI-agent — demonstrasjon inkludert"
- Mål: 3–5 betalende pilotkunder

### Fase 3: Skaler (mnd 6–12)
- Refinere tilbudet basert på learnings
- Begynn å trekke inn produktidé fra fase 2-data
- Potensielt ansette første menneske (selger/administrativ)
